## 修改方式

- **改文案**：在 `src/data/scenarios.ts` 找到對應 `stage(n, [ ... ])` 的該則情境，改 `title`、`description`、各 `options[].label`。
- **改數值**：改該則情境的 `options[].effect`（如 `attributes`、`lifeStatus`、`riasec`、`scandalDelta`、`triggers` 等）。
- **改門檻**：在對應 `options[]` 加上 `requirements: { minAttributes: { I: 50 } }` 等。
- **總回合數**：`GameScreen.tsx` 內 `TOTAL_TURNS = 20`；若要玩滿 50 題可改為 50，並配合 `PATH_LENGTH`（地圖路徑格數）調整。

結局邏輯與 20 重結局矩陣見 **`src/engine/gameEngine.ts`** 的 `evaluateEnding()` 與 **`GAME_DESIGN.md`**。
